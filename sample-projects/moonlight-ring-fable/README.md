# 寓話サンプルプロジェクト

Scenario Studio の検証用に作成した、短編の寓話モチーフ作品です。

モチーフは有名寓話「ウサギとカメ」ですが、舞台を近未来の空中都市へ置き換えています。

## 作品コンセプト

- **元ネタ**: ウサギとカメ
- **設定変更**: 徒競走ではなく「空中都市の夜間配送レース」
- **驚きの展開**: 勝利条件が仕様書の 3 ページ目に書いてあった。

## 開き方

1. `pnpm -F @scenario-studio/frontend dev` で dev サーバ起動
2. ブラウザで http://localhost:5173/
3. 「既存プロジェクトを開く」 → `sample-projects/moonlight-ring-fable/` を選択

## 入っているもの

| 種類 | 数 | 例 |
|---|---:|---|
| Character | 3 | ラピ(兎) / ノロ(亀) / クロウ(実況) |
| Location | 2 | スカイリング中央 / 風切り高架 |
| Item | 1 | ルーメン・バトン |
| Chapter / Scene | 1 / 3 | 挑戦 / 激走 / 月光の判定 |
| Glossary | 3 | 月光リング便 / ルーメン・バトン / 安全優先プロトコル |
| Relation | 3 | ライバル / 相棒 / 観察対象 |
